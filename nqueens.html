Descrição

O problema das n rainhas é o problema matemático de dispor n rainhas em um tabuleiro de xadrez de dimensão nxn, de forma que nenhuma delas seja atacada por outra. Para tanto, é necessário que duas damas quaisquer não estejam numa mesma linha, coluna, ou diagonal. Mais info.

The n queens puzzle is the problem of placing n chess queens on an nxn chessboard so that no two queens threaten each other. Thus, a solution requires that no two queens share the same row, column, or diagonal. More info.
